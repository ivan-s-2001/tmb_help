# Too Many Bones Helper

Второй рабочий этап стартового экрана.

Что уже есть:
- стартовая страница выбора персонажа;
- визуальный слой на пользовательских ресурсах;
- реальные арты персонажей встроены в карточки;
- базовая структура для будущих игровых материалов;
- кубики пока не подключены в интерфейс.

## Запуск

```bash
npm install
npm run dev
```
