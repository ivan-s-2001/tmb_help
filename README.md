# Too Many Bones Helper

Минимальный стартовый этап нового сайта с нуля.

Сейчас в проекте только один рабочий экран:
- стартовая страница;
- выбор персонажа;
- тёмный mobile-first интерфейс;
- простой устойчивый каркас без лишней логики.

## Запуск

```bash
npm install
npm run dev
```
